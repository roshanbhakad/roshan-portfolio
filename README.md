# Roshan Bhakad - Developer Portfolio

A premium, interactive, and high-fidelity developer portfolio website showcasing my qualifications as an AI & ML Engineer and Mechatronics Enthusiast.

## 🚀 Live Preview
The website features a futuristic dark-mode cybernetic theme with neon cyan and purple accents.

- **Responsive Navigation**: Sticky glassmorphic navbar with mobile menu compatibility.
- **Dynamic Hero Section**: Real-time typing effects highlighting my specializations and call-to-actions (CTAs) for downloading my resume.
- **Interactive Canvas Particles**: A custom-built background animation that reacts to mouse hover and movement.
- **Technical Skills Hub**: Structured categorization of my engineering capabilities (Programming, AI/ML, Databases, DevOps, Tools).
- **Project Showcases**: Glowing dashboard cards displaying my major hardware & software systems.
- **Experience & Education Timelines**: Sleek vertical and horizontal history tracks.
- **Contact Form**: Interactive user feedback form with mock submit processing.

---

## 🛠️ Tech Stack & Features
- **Frontend Core**: Vanilla HTML5, JavaScript (ES6+)
- **Styling**: Vanilla CSS3 (Custom variables, glassmorphism, responsive keyframe animations)
- **Interactive Elements**: Canvas API, Intersection Observer API
- **Fonts**: Outfit & Space Grotesk (Google Fonts)

---

## 📂 Project Structure
- `index.html` - The structural markup and semantic sections.
- `styles.css` - Custom design system, typography, timelines, and animations.
- `app.js` - Particle engine, typing effects, and viewport observer loops.
- `avatar.jpg` - Profile headshot.
- `resume.pdf` - Downloadable curriculum vitae.

---

## 💻 Running Locally

To run the portfolio website locally, clone this repository and spin up a lightweight server in the root directory:

**Using Python:**
```bash
python -m http.server 3000
```
Then open your browser and navigate to `http://localhost:3000`.
